# Remember

Simple command line task manager.

## Commands

- help, h -> Show help.
- get, g, (empty)   -> Get all tasks.
- show, s [Task ID] -> Show task info.
- add, a [Task]     -> Add a new task.
- delete, del, d <Task IDs ...>  -> Delete task.
- finish, f <Task IDs... > -> Finish task.

## Add

- -f DD/MM/YY - End date.
